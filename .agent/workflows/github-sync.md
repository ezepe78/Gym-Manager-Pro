---
description: Realiza un commit de todos los cambios y los sincroniza con el repositorio remoto de GitHub
---

Para guardar tus cambios y subirlos a GitHub:

1. Ejecuta `git add .` para añadir todos los archivos modificados al área de preparación.
2. Ejecuta `git commit -m "Actualización: Sincronización de cambios"` para crear el commit con un mensaje descriptivo.
3. Ejecuta `git push` para enviar los cambios a la rama principal en GitHub.
